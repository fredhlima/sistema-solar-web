# Sistema Solar — Simulador Interativo (showcase web)

Simulador 3D didático do Sistema Solar: 43 corpos, missões espaciais, eventos astronômicos, quiz e tour guiado. Em português, inglês e espanhol.

**▶ Rode no navegador:** https://fredhlima.github.io/sistema-solar-web/

Este repositório é o espelho público **somente do app web** (showcase). O desenvolvimento acontece em repositório privado; este espelho é atualizado a partir dele e não recebe pull requests.

Feito com Three.js, JavaScript puro e física de Kepler validada contra dados da NASA (NSSDC/JPL).

Política de privacidade: [privacidade.html](https://fredhlima.github.io/sistema-solar-web/privacidade.html)
